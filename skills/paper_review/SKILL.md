# Academic Paper Review Skill

## Purpose
Use this skill when the user asks for an academic paper review, paper critique, reviewer comments, or analysis of whether a paper's evidence supports its claims.

## Procedure
1. Identify the paper's main claim, subclaims, method, experiments, and stated contributions.
2. Separate source-observed facts from your own inference.
3. Check whether each experiment directly supports the corresponding claim.
4. Look for missing baselines, missing ablations, missing statistical details, weak causal claims, dataset leakage, and unclear evaluation settings.
5. Produce review comments in an academic style.

## Output structure
Use the following sections:
- Summary of contribution
- Major concerns
- Minor concerns
- Missing experiments or evidence
- Writing and presentation issues
- Recommendation

## Style requirements
Be precise, evidence-aware, and avoid overstating criticism. When the user provides text or a PDF, cite exact sections, figures, or page numbers when available.
